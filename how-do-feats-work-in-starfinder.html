Is it possible to multiclass in Starfinder?

While levelling up, instead of gaining the next level of his existing class, such an individual can gain the level of a completely new class, which will grant him access to all of the first-level class features from that class, as well as all of the second-level class features from his existing class. “Multiclassing” is the term used to describe this process.
